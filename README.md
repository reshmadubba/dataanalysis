# Exploratory Data Analysis (EDA) Project

## Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on a dataset to understand its structure, identify patterns, detect missing values, analyze relationships between variables, and generate meaningful insights using statistical methods and visualizations.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- VS Code

## Features
- Load dataset from CSV file
- Display first few rows of data
- Show dataset information and statistics
- Detect missing values
- Generate histograms for data distribution
- Create correlation heatmaps
- Detect outliers using boxplots
- Generate pairplots for feature relationships
- Provide data insights and conclusions

## Conclusion
This project helps in understanding datasets through visualization and analysis, making it easier to identify trends and patterns before applying machine learning techniques.
